# 🌱 Biota Protocol: Suelo Vivo & UBI Regenerativo

> **Transformando la salud del suelo en activos digitales y bienestar campesino sobre Celo.**

Biota Protocol es una plataforma de **Finanzas Regenerativas (ReFi)** que utiliza Inteligencia Artificial Multimodal y Tecnología Blockchain para empoderar a los productores rurales. El protocolo permite diagnosticar, certificar y monetizar la regeneración de suelos en tiempo real.

## 🚀 Características Principales

- **Sistema Multi-Agente (Gemini 1.5 Pro):** Orquestador de IA especializado en diagnósticos de cromatografía y análisis microbiológico.
- **BiotaPass (NFT Dinámico):** Un pasaporte biológico on-chain que evoluciona a medida que mejora el Bio-Score del suelo.
- **Proof of Action (PoA):** Certificación geográfica y temporal de labores agrícolas (como la aplicación de Microorganismos de Montaña).
- **Renta Básica Regenerativa (UBI):** Distribución automatizada de incentivos en **cUSD** vía MiniPay para productores certificados.

## 🛠️ Stack Tecnológico

- **Frontend:** Next.js 15 (App Router) + Tailwind CSS.
- **Web3:** Privy Auth, Wagmi, Viem.
- **Network:** Celo Sepolia (Testnet) / Celo Mainnet.
- **IA:** Google Gemini 1.5 Pro API.
- **Mobile:** Optimizado para Telegram Mini Apps (TWA).

## 🔒 Privacidad y Propiedad Intelectual

Biota Protocol protege el conocimiento técnico ancestral y técnico:

- **Lógica Protegida:** Los algoritmos de análisis de cromatogramas y las fórmulas exactas de los Microorganismos de Montaña (MM) residen en el **Backend (Server-Side)** y no son expuestos públicamente.
- **Seguridad:** Las llaves privadas y credenciales de API están protegidas mediante variables de entorno y no se sincronizan en este repositorio.

## 📦 Instalación

1. **Clonar:** `git clone https://github.com/tu-usuario/biota-scrow.git`
2. **Dependencias:** `npm install`
3. **Variables:** Copiar `.env.example` a `.env.local` y completar las llaves.
4. **Ejecutar:** `npm run dev`
